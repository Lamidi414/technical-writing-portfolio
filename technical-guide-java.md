# How to Install Java and Run Your First Program

## Overview
This guide explains how to install Java on your system and run a simple Java program.

---

## Step 1: Download Java

1. Go to the official Java website:
   https://www.oracle.com/java/technologies/downloads/
2. Download the latest version of JDK (Java Development Kit).

---

## Step 2: Install Java

1. Open the downloaded file.
2. Follow the installation steps.
3. Complete the installation.

---

## Step 3: Verify Installation

Open Command Prompt and run:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
### Compile the Program
```bash
javac HelloWorld.java
```
### Run the program
```bash
java HelloWorld
```

## Output
```bash
Hello, World!
```
